# Useful-Bookmarklets

A collection of useful scripts that can be bookmarked and executed.

### Instructions:

1. Go find a bookmarklet script

    - [UTexas Chemistry](/UTexas-Chemistry/)
    - [Delta Math](/DeltaMath/)

2. Create a new bookmark, you can achieve this in several different ways

    - Right clicking on the bookmark bar and adding a new bookmark
    - Hitting Ctrl + D to favorite current site and edit the bookmark
    - A browser's bookmark manager

3. Set a name for the new bookmark

4. Copy the bookmarklet script and paste it into the URL field

5. Go to the site where the script is modifying and just click on saved bookmark.
    - If you cannot see the bookmark bar, press <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>B</kbd>
